title: "Patterns"
---

Common UX patterns implemented with htmx. Copy and adapt for your project.

## Loading

- [Click to Load](/patterns/click-to-load) - Load content when you click an element
- [Infinite Scroll](/patterns/infinite-scroll) - Load content when you scroll to bottom
- [Lazy Load](/patterns/lazy-load) - Load content after the page renders
- [Progress Bar](/patterns/progress-bar) - Show progress bar during background job

## Forms

- [Active Search](/patterns/active-search) - Filter search results as you type
- [Active Validation](/patterns/active-validation) - Validate form input as you type
- [Linked Selects](/patterns/linked-selects) - Update select options via another select
- [File Upload](/patterns/file-upload) - Upload files with progress and validation
- [Reset on Submit](/patterns/reset-on-submit) - Clear form inputs after submission

## Records

- [Edit in Place](/patterns/edit-in-place) - Update a record without page refresh
- [Delete in Place](/patterns/delete-in-place) - Remove a record without page refresh
- [Bulk Actions](/patterns/bulk-actions) - Perform actions on multiple records
- [Drag to Reorder](/patterns/drag-to-reorder) - Change order of records with drag and drop
- [Edit Row](/patterns/edit-row) - Edit table rows inline

## Display

- [Dialogs](/patterns/dialogs) - Show modals and popups on demand
- [Animations](/patterns/animations) - Animate content as it swaps in
- [Tabs](/patterns/tabs) - Switch between content panels using tabs
- [Confirm](/patterns/confirm) - Show confirmation before actions

## Advanced

- [Update Other Content](/patterns/update-other-content) - Update multiple elements on the page at once
- [Keyboard Shortcuts](/patterns/keyboard-shortcuts) - Bind keyboard shortcuts to interactive elements
